## Role

Mobile versions of application.

## Install

```
npm install cordova -g
```

## Setup

You need to add a platform for which you want to build your app.
For example:

```
cordova platform add browser
```

## Run

```
# application is already running (foreman start)

cd servers/cordova
cordova run browser
```
